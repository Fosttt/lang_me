# LangMe

Личное бесплатное приложение для изучения английских слов в стиле BigBean:
вертикальная лента карточек, интервальные повторения (SM-2), проверка
произношения, тепловая карта прогресса — плюс AI-фичи через собственный
LLM-сервер (подписка Claude Code, модель haiku).

Всё офлайн, кроме AI-фич. Прогресс хранится только на устройстве.

## Структура

- `app/` — Flutter-приложение (Android).
- `server/` — LLM-прокси для VPS: FastAPI поверх `claude -p --model haiku`.
- `tools/` — генератор базы слов, валидатор, патч Android-манифеста для CI.

## Как получить APK

Локальный Android SDK не нужен — собирает GitHub Actions:

1. Пушим в `main` (или запускаем workflow `build-apk` вручную).
2. Забираем APK: вкладка **Actions → build-apk → artifact `lang_me-apk`**,
   либо релиз **latest** на странице Releases.
3. Ставим APK на телефон (разрешить установку из неизвестных источников).

Папка `app/android/` не хранится в репозитории — CI генерирует её через
`flutter create` и патчит манифест скриптом `tools/patch_android.py`
(разрешения INTERNET и RECORD_AUDIO, `<queries>` для распознавания речи и TTS).

## Что умеет приложение

- **Лента** — полноэкранные карточки слов (свайп вверх): слово, транскрипция,
  перевод, пример с переводом, озвучка, кнопки «Знаю»/«Учить»/закладка.
  Визуал — эмодзи темы на градиенте; можно подложить свои картинки в
  `app/assets/images/<слово>.jpg` до сборки.
- **Placement-тест** при первом запуске — определяет уровень CEFR (A1–C1).
- **Повторение** — SM-2 (как в Anki): «снова / трудно / хорошо / легко»,
  бейдж с числом слов на сегодня. Интервал ≥ 21 дня = слово «освоено».
- **Тренировка** — 4 режима: EN→RU, RU→EN, сборка слова из букв, аудирование.
- **Произношение** — микрофон + офлайн-распознавание, вердикт по расстоянию
  Левенштейна.
- **Диалоги** — у каждого слова мини-диалог из 4 реплик (говорящие A/B, два
  голоса TTS разной высоты), кнопка 💬 на карточке и в словаре. Перевод
  включается/выключается, «прослушать весь диалог» с подсветкой текущей
  реплики; режим практики — ты озвучиваешь реплики собеседника B в микрофон
  и получаешь оценку. Диалоги вшиты в базу (`tools/generate_dialogs.py`);
  для слова без диалога он сочиняется на лету через AI-сервер
  (эндпоинт `/llm/dialog`, кэшируется на сервере и в приложении).
- **Словарь** — поиск по базе, закладки, свои пометки к словам.
- **Профиль** — тепловая карта активности (18 недель), стрик, статистика по
  статусам и уровням.
- **AI (через мой сервер)** — «объясни иначе», «ещё примеры», проверка моего
  предложения, чат-репетитор, который вплетает недавние слова. Ответы
  explain/examples кэшируются локально. Без сети и без настроенного сервера
  эти кнопки скрыты, всё остальное работает офлайн.
- **Настройки** — уровень, тёмная тема, автоозвучка, скорость TTS, дневная
  цель, URL+токен сервера (с проверкой соединения), экспорт/импорт прогресса
  через буфер обмена, полный сброс.

## База слов

`app/assets/words.json` — стартовые ~150 слов. Пополнение через подписку
Claude Code (на сервере с залогиненным `claude`):

```bash
./tools/generate_words.sh A2 3    # 3 батча по 100 слов уровня A2
./tools/generate_words.sh all 2   # по 2 батча каждого уровня A1..C1
```

Скрипт спрашивает у haiku новые слова (исключая уже имеющиеся), валидирует
JSON (`tools/validate_words.py`, битый батч повторяется до 3 раз) и вливает
в базу. После пополнения — коммит и пуш: CI соберёт APK с новой базой.

## Сервер AI-фич (VPS)

```bash
cd server
python3 -m venv venv && venv/bin/pip install -r requirements.txt
cp env.example .env   # вписать длинный случайный LANGME_TOKEN
sudo cp langme-llm.service /etc/systemd/system/
sudo systemctl daemon-reload && sudo systemctl enable --now langme-llm
sudo ufw allow 8977/tcp
```

Требования: на сервере установлен и залогинен Claude Code CLI (подписка).
Сервис слушает порт 8977, авторизация — заголовок `X-Auth-Token`, один
одновременный LLM-запрос (иначе 429), ответы explain/examples кэшируются в
`~/.cache/langme_llm`. Модель — haiku (дёшево для лимитов подписки).

В приложении: Профиль → Настройки → AI-сервер → URL `http://<IP VPS>:8977`
и токен из `.env`, кнопка «Проверить».
